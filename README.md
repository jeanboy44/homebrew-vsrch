# Homebrew Tap for vsrch

This repository publishes a Homebrew formula for installing `vsrch`.

## Install

```bash
brew tap jeanboy44/vsrch
brew install vsrch
```

To upgrade:

```bash
brew update
brew upgrade vsrch
```

## How the formula is generated

The formula is generated by the main repo and synced via:

```bash
GIT_TAG=<tag> just sync-brew-tap tap_dir=/path/to/homebrew-vsrch
```

Example:

```bash
cd ~/workspace/vsrch
GIT_TAG=v0.1.1 just sync-brew-tap tap_dir=/Users/jeanboy/workspace/homebrew-vsrch
```

It points to release artifacts on GitHub:

- `vsrch-<tag>-macos-latest-*.tar.gz`
- `vsrch-<tag>-ubuntu-latest-*.tar.gz`

Each package includes:

- `bin/vsrch`
- `data/models` (when packaged with models)
- `README.md`
- `INSTALL.md`
